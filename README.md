#### ✅ 프로젝트 소개

`넷플릭스` 웹사이트를 복제하며 `리액트`의 심화적인 기능을 사용해봤습니다.

#### ✅ 구현 기능 목록

1. `콘텐츠 나열`: Row 컴포넌트, 카드 이미지, 터치 슬라이드
2. `메인 이미지`: 최신 영화 랜덤 출력, 예고편 보기
3. `상세 페이지`: 모달 창, 영화 제목/줄거리/평점 불러오기
4. `검색 기능`: 제목 검색, 검색 결과 페이지, 관련 정보 스틸컷 띄우기
