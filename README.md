#### ✅ 프로젝트 소개

`넷플릭스` 웹사이트를 복제하며 `리액트`의 심화적인 기능을 사용해보았습니다.

#### ✅ 구현 기능 목록

1. Row 컴포넌트, 카드 이미지, 터치 슬라이드
2. 최신 영화 메인으로 출력, 예고편 보기
3. 모달, 상세 정보 불러오기
4. 검색 기능 및 페이지
